    2. Project

    Number Guessing Game

    Description

    The Number Guessing is a fun and interactive project designed to test and challenge a player's ability to guess a randomly generated number within a specified range. the game provides instant feedback, helping players refine their guesses and improve their problem-solving skills.

    Key Features:

    1.Random Number Generations

  • The program generates a random number within a predefined range(e.g., 1 to 10000).

    • 2. User Interaction:

  • Player input their guesses, and the game provides feedbacksuch as "Too High" or "Too Low" to guide them

    • 3. Score Tracking

  • Tracks the number of attempts taken by the player to guess the correct the number.

    • 4. Difficuity Levels:

  • Offer multiple levels of difficulty by adjusting the range of numbers or limiting the numbers of attempts.

    • 5. Replay Option

  • Allowss players to restart the game and try again.

    • Technologies Uesd:

  • Frontend: HTML, CSS, JavaScript.
  • Implementation:JavaScript for number generation and user interaction

    • Objective:

    To create a simple yet engaging game that demonstrate the use of programming logic, conditional statement, and user input handling in a web-based enviroment.

    Outcome

    The project successfully delivers an entertaining and eductional experiences for user, showcasing the power of JavaScript in creating interactive applications

    Future Enhancement

  • Adding a leaderboard to record high scores
  • Intergating a timer to make the game more competitive
  • Expanding the game to include multiplayer functionality.
